# Minigolf - Clase 6

Alumno: Federico Bazán
Fecha: 25-3-26

---

Descripción

Juego de minigolf en 3D con 2 niveles. Controlás la dirección y fuerza del golpe para embocar la pelota.

---

Controles

- A/D: Rotar dirección
- Click (mantener): Cargar fuerza
- Soltar: Golpear
- Espacio: Resetear posición

---

Ejercicios

Ejercicio 1: Reset con Espacio
- Script: ResetBola.cs
- La bola vuelve al spawn al soltar Espacio

Ejercicio 2: Medir Tiempo de Click
- Script: GolpearBola.cs
- Cuenta cuánto tiempo se mantiene presionado el mouse

Ejercicio 3: Fuerza según Tiempo
- Script: GolpearBola.cs
- Más tiempo presionado = más fuerza del golpe
- Máximo: 3 segundos

Ejercicio 4: Trigger para Cambiar Escena
- Script: HoyoTrigger.cs
- Cuando la bola cae en el hoyo, cambia al siguiente nivel
- En Nivel 2 muestra "¡GANASTE!"

Ejercicio 5: Prototipo Completo
- 2 niveles jugables
- Mapas creados en Blender (esto no valió la pena profe, mucho tiempo perdido tratando de entender esta herramienta jaja)
